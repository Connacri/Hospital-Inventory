# 🏥 Hospital Inventory Management System

![Status](https://img.shields.io/badge/Status-Stable-success?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20Windows-blue?style=for-the-badge)

Une solution complète et performante pour la gestion des stocks hospitaliers. Ce système permet un suivi rigoureux des ressources médicales, garantissant une organisation optimale pour les établissements de santé.

🔗 **Consulter la documentation :** [Hospital Inventory Web](https://connacri.github.io/Hospital-Inventory/)

---

## 🛠️ Guide d'installation

### 🖥️ Windows
1. Téléchargez `installer.exe`.
2. Exécutez le fichier et suivez l'assistant d'installation.
3. Lancez l'application depuis votre menu Démarrer.

### 📱 Android
1. Téléchargez `app-release.apk`.
2. Si votre navigateur le demande, autorisez l'installation depuis cette source.
3. Ouvrez le fichier pour finaliser l'installation sur votre smartphone.
